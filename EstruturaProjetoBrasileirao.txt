Explicação da Estrutura

app/core:

config.py: Arquivo para definir configurações globais, como banco de dados e CORS. 
database.py: Criação do Session e inicialização do banco usando SQLAlchemy.


app/models:

Contém os modelos SQLAlchemy relacionados às tabelas no banco. Cada tabela é definida em um arquivo separado para organização e manutenibilidade.


app/routes/crud:

Rotas específicas para as operações CRUD relacionadas às tabelas.
Estruturado em arquivos separados para cada entidade.


app/routes/endpoints:

Rotas especializadas por funcionalidade/tela. Por exemplo, login.py cuida apenas do login, enquanto cadastro.py pode centralizar os endpoints de diversos cadastros.


app/services:

Camada separada para regras de negócio. Aqui você implementa funcionalidades complexas ou validações além das rotas simples.
Exemplo: Verificação se um clube já existe antes de criar um novo registro.


app/schemas:

Schemas Pydantic para validação e serialização de dados nos endpoints. Exemplo: validação de entrada de dados de usuários ou cidades.


app/utils:

Funções utilitárias, como:Hash de senhas usando bcrypt (ex. hashing.py).
Gerenciamento de JWT para autenticação (ex. token.py).


app/main.py:

Entrypoint do projeto. Configura FastAPI, conecta o banco e inclui as rotas.

4. routes/crud/clube.py

from fastapi import APIRouter, Depends, HTTPException
from sqlalchemy.orm import Session

from app.models.clube import Clube
from app.schemas.clube import ClubeCreate
from app.core.database import SessionLocal
from app.services.clube_service import verificar_nome_clube_unico

router = APIRouter()

def get_db():
    db = SessionLocal()
    try:
        yield db
    finally:
        db.close()

@router.post("/")
def criar_clube(clube: ClubeCreate, db: Session = Depends(get_db)):
    if not verificar_nome_clube_unico(db, clube.nome):
        raise HTTPException(status_code=400, detail="Clube já cadastrado.")
    novo_clube = Clube(nome=clube.nome, cidade_id=clube.cidade_id)
    db.add(novo_clube)
    db.commit()
    db.refresh(novo_clube)
    return novo_clube
